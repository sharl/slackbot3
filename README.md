slackbot3
=========

sample script for slackbot

## preinstall
```
git clone https://github.com/sharl/geeklets.git
cp geeklets/.amedas ~
cp geeklets/{amedas,amesh} ~/bin
```

```
pip install --upgrade slack_sdk aiohttp requests bs4
```

$ SLACK_TOKEN=xoxb-hogehoge ./slackbot3.py
